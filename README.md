# 🐂 NTX-Game: Bull Arena (疯牛竞技场)

> "在疯牛的冲撞下，你能幸存吗？"

这是一个基于区块链的博弈生存游戏。

## 🎮 游戏规则

1.  **回合制**: 每一轮持续 **1 分钟**。
2.  **下注**: 玩家选择 **8 个牛栏 (Barns)** 中的一个，并存入 **NTX 代币**。
3.  **锁定**: 倒计时结束后，停止下注。
4.  **袭击**: 疯牛 (The Mad Bull) 随机冲撞其中一个牛栏。
5.  **结算**:
    *   💥 **被撞牛栏**: 玩家资金全部损失。
    *   🛡️ **幸存牛栏**: 玩家拿回本金 + 瓜分被撞资金的 **99.5%**。
    *   💰 **奖励池**: 0.5% 自动转入社区奖励池。

## 🎨 视觉设计理念：赛博斗兽场 (Cyberpunk Arena)

> 拒绝简约，追求极致的视觉冲击与沉浸感。

### 1. 核心风格
*   **主题**: **机械狂潮 (Mecha-Frenzy)**
*   **色调**: 
    *   背景: 深邃黑 (#050505) + 雾霾蓝 (#0f172a)
    *   强调色: 霓虹红 (#ff0055) - 危险/疯牛
    *   辅助色: 赛博黄 (#fcee0a) - 财富/警告
    *   安全色: 全息绿 (#00ff9d) - 幸存/收益
*   **材质感**: 金属拉丝、全息投影、蒸汽烟雾、破碎玻璃。

### 2. 沉浸式场景设计
*   **背景**: 动态的地下斗兽场，四周是全息投影的观众席，不断有闪光灯和呐喊声浪的可视化波纹。
*   **牛栏 (The Barns)**: 
    *   不再是简单的方块，而是 **8 个带有高压电网的金属囚笼**。
    *   **状态变化**:
        *   *空闲*: 蓝色的全息编号悬浮在上方。
        *   *下注*: 玩家投入代币后，笼子顶部亮起金色聚光灯，显示当前奖池金额。
        *   *被毁*: 笼子扭曲变形，电火花四溅，烟雾弥漫。
*   **主角: 机械疯牛 (Mecha-Bull)**:
    *   一只由废弃引擎和液压臂组装成的机械巨兽。
    *   **待机**: 在屏幕中央的黑暗处，只能看到两只发红光的眼睛和喷出的蒸汽。
    *   **袭击**: 伴随屏幕剧烈震动，疯牛冲出，拖着残影撞向目标。

### 3. 动画与特效 (FX)
*   **Lottie/Rive 动画**: 用于复杂的牛奔跑和撞击动作，保证流畅度。
*   **粒子系统 (Particles)**: 
    *   下注时：代币化作流光飞入笼子。
    *   撞击时：碎片、火星、烟尘爆炸效果。
    *   结算时：金币雨 (Coin Rain) 覆盖屏幕。
*   **屏幕震动 (Screen Shake)**: 撞击瞬间，整个网页跟随震动，增强打击感。
*   **音效设计 (Sound Design)**:
    *   背景音: 低沉的工业电子乐 (Industrial Techno)。
    *   倒计时: 类似心脏跳动的急促警报声。
    *   撞击: 金属碰撞的巨响。

## 🛠 技术架构

*   **智能合约**: Solidity (Ethereum/EVM Compatible)
*   **随机数**: Chainlink VRF (生产环境) / Keccak256 (测试环境)
*   **前端**: Next.js + TailwindCSS + Framer Motion + RainbowKit

## 📂 目录结构

*   `contracts/`: 智能合约源码 (Hardhat)
    *   `MockNTX.sol`: 测试用代币
    *   `BullArena.sol`: 游戏核心逻辑
*   `frontend/`: 游戏前端界面 (Next.js)
